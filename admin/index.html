How to use it

For projects, the normal flow is simple: add the title, paste the website URL, choose where it should show, then save.

Default pages talk about building the website. Marketing talks about handling marketing support.

If image is empty, the public card tries the website screenshot first and falls back to the default image if it cannot load.

If you preview via a static server like Live Server, the dashboard can open but saving needs PHP.
